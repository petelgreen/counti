# Project Rules

All user requests must first be handled by the Manager Agent.

The Manager Agent is responsible for:

- understanding the request
- breaking it into tasks
- assigning the correct agent
- orchestrating execution

- Keep the MVP small
- Prefer simple and explicit solutions
- Do not invent requirements without stating assumptions
- AI results must be editable
- Do not return raw freeform AI output directly to users
- Store original input and parsed result
- Always surface uncertainty when relevant
- Optimize for mobile-first UX
